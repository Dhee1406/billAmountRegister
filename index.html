About the project

This Cash Register Manager application helps with the bill and cash management system. It enables a user to input a bill amount and cash given, then calculates the change to be returned or notifies if the Input is valid or not. It also displays error messages and presents the quantity of specific denominations of the change to be returned using the available Indian currency notes (2000, 500, 200, 100, 50, 20, 10, 5, 2, 1).
